# 地图编辑器

地图编辑器用于创建和编辑游戏地图。

## 打开地图编辑器

1. 确保在「游戏模式」下
2. 点击菜单栏的「地图」按钮
3. 地图编辑器会在主区域显示

## 瓦片集（Tileset）

瓦片集是一张包含多个小瓦片的图片，用于绘制地图。

### 导入瓦片集

1. 在资源面板点击「地图」标签
2. 点击「瓦片集」子标签
3. 点击「+」导入瓦片集图片
4. 设置瓦片大小（如 32x32 像素）

- 室内瓦片集

## 绘制地图

### 选择瓦片

1. 在右侧面板选择瓦片集
2. 点击要使用的瓦片
3. 瓦片会高亮显示

### 绘制操作

- **左键点击**：放置瓦片
- **右键点击**：删除瓦片
- **拖动**：连续绘制

## 图层管理

地图可以包含多个图层，用于分隔不同类型的元素。

### 创建图层

1. 在图层面板点击「+」
2. 输入图层名称
3. 设置图层顺序

### 图层类型

| 图层 | 用途 |
|------|------|
| 背景层 | 装饰性元素，无碰撞 |
| 地面层 | 角色可以站立的地面 |
| 碰撞层 | 设置碰撞区域 |
| 前景层 | 遮挡角色的前景元素 |

### 图层顺序

- 数字越大越在前面
- 建议：背景(0) → 地面(1) → 碰撞(2) → 前景(3)

## 碰撞设置

### 标记碰撞区域

1. 选择「碰撞层」
2. 使用瓦片标记碰撞区域
3. 碰撞瓦片会以特殊颜色显示

### 碰撞标签

为碰撞区域设置标签，便于在代码中识别：

```python
# 检测是否碰到 "地面" 标签的区域
if hero.is_touch("地面"):
    print("站在地面上！")
```

## 保存和使用

### 保存地图

点击「保存」按钮，地图会保存为 `.bgm` 文件。

### 在代码中加载地图

```python
# 加载地图
load_map("我的地图")

# 创建角色
hero = Sprite("玩家")

# 镜头跟随
follow(hero)
```

## 常见问题

### Q: 瓦片显示不正确？

1. 检查瓦片大小设置是否正确
2. 确认瓦片集图片格式为 PNG
3. 尝试重新导入

### Q: 碰撞检测不工作？

1. 确保使用了碰撞层
2. 检查碰撞标签是否正确
3. 使用 `show_collision(hero)` 查看碰撞范围

### Q: 地图太大导致卡顿？

1. 减少同屏显示的瓦片数量
2. 使用更小的瓦片尺寸
3. 优化不必要的图层
