# 角色编辑器

角色编辑器用于创建和编辑游戏角色（精灵）。

## 打开角色编辑器

1. 确保在「游戏模式」下
2. 点击菜单栏的「角色」按钮
3. 角色编辑器会在主区域显示

## 角色包（.bgs）

BingoCodeIDE 使用 `.bgs` 格式的角色包文件。一个角色包包含：

- **config.json**：角色配置文件
- **PNG 图片**：角色的每一帧图片

### 创建新角色

1. 在资源面板点击「角色」标签
2. 点击「+」按钮
3. 选择一张或多张 PNG 图片
4. 输入角色名称
5. 点击确定

## 帧动画

帧动画是通过快速切换多张图片来实现的动画效果。

### 添加动画帧

1. 在角色编辑器中打开角色
2. 点击「动画」标签
3. 点击「+」添加新帧
4. 选择图片文件

<!-- screenshot: editor/animation-frames.png -->
<!-- alt: 动画帧编辑 -->
<!-- description: 显示动画帧列表和预览 -->

### 设置帧速率

帧速率控制动画播放速度：

- **每帧时间**：每张图片显示的秒数
- 默认值：0.1 秒（每秒 10 帧）
- 值越小，动画越快

### 动画命名

为动画命名以便在代码中调用：

```python
# 播放名为 "walk" 的动画
hero.play("walk")
```

## 碰撞框

碰撞框决定了角色的碰撞检测范围。

### 查看碰撞框

1. 在角色编辑器中打开角色
2. 点击「碰撞」标签
3. 碰撞框会以半透明颜色显示

<!-- screenshot: editor/collision-box.png -->

### 调整碰撞框

- 拖动边缘调整大小
- 拖动中心调整位置
- 点击「重置」恢复默认

### 碰撞框大小倍率

在代码中可以调整碰撞框大小：

```python
hero.hitbox_scale = 0.8  # 碰撞框缩小到 80%
```

## 属性设置

### 基本属性

| 属性 | 说明 | 默认值 |
|------|------|--------|
| 名称 | 角色在代码中的名字 | 必填 |
| 大小 | 初始缩放百分比 | 100 |
| 朝向 | 初始角度 | 0 |
| 图层 | 渲染顺序 | 1000 |

### 旋转模式

| 模式 | 说明 |
|------|------|
| 自由旋转 | 角色可以任意角度旋转 |
| 左右翻转 | 只水平翻转，不旋转图片 |
| 不旋转 | 完全不旋转 |

## 保存和导出

### 保存角色

点击「保存」按钮，角色会保存为 `.bgs` 文件。

### 导出图片

可以将角色导出为单张 PNG 图片。

## 常见问题

### Q: 角色导入后显示不正常？

1. 检查图片格式是否为 PNG
2. 确认图片尺寸是否合适（建议 64x64 或 128x128）
3. 尝试重新导入

### Q: 动画播放速度不对？

调整帧速率设置。每帧时间越短，动画越快。

### Q: 碰撞检测不准确？

1. 检查碰撞框是否正确
2. 使用 `show_collision(hero)` 查看实际碰撞范围
3. 调整 `hitbox_scale` 属性
