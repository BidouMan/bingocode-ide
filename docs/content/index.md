# 欢迎使用 BingoCodeIDE

BingoCodeIDE 是一款专为教学设计的 2D 游戏开发 IDE，让老师和学生能够轻松创建属于自己的游戏。

## 为什么选择 BingoCodeIDE？

- **简单易学**：使用 Python 编写游戏逻辑，语法简洁直观
- **可视化编辑**：内置角色编辑器和地图编辑器，无需第三方工具
- **实时预览**：编写代码的同时就能看到游戏效果
- **内置资源**：提供丰富的角色和地图素材，开箱即用
- **AI 辅助**：集成 AI 编程助手，遇到问题随时提问

## 适用场景

| 场景 | 说明 |
|------|------|
| 课堂教学 | 老师可以用它教授编程基础和游戏设计 |
| 学生自学 | 零基础学生也能跟着教程做出第一个游戏 |
| 创客教育 | 快速原型开发，验证游戏创意 |
| 编程入门 | 通过游戏学习 Python 编程 |
